# Forge resource type icons (Holocron-style)

Place Holocron Toolset PNG icons here, named by icon id (e.g. `k1_2da.png`, `k1_blank.png`, `k1_sound.png`).

Icon ids are defined in `src/apps/forge/data/ForgeResourceIcons.ts` (`RESOURCE_ICON_BY_EXTENSION` and `getResourceIconId()`). The UI requests paths like `assets/icons/k1_2da.png`.

If you have the Holocron assets under another path (e.g. workspace `assets/` with long filenames), copy or symlink them here with the short names: `k1_2da.png`, `k1_creature.png`, `k1_dialog.png`, `k1_door.png`, `k1_git.png`, `k1_item.png`, `k1_journal.png`, `k1_model.png`, `k1_none.png`, `k1_placeable.png`, `k1_script.png`, `k1_sound.png`, `k1_soundset.png`, `k1_tlk.png`, `k1_trigger.png`, `k1_walkmesh.png`, `k1_waypoint.png`, `k2_texture.png`, and `k1_blank.png` for fallback.
